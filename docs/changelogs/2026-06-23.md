# 2026-06-23 更新日志

## 修复

### Nginx 崩溃 (Exited 255)
- **根因**: `nginx.conf` 中 `minio:9000` 和 `weclaw-st-*:8000` 的 `proxy_pass` 使用硬编码 hostname，在目标容器就绪前 nginx 解析 DNS 失败导致 emerg 退出
- **修复**: 改为变量方式，利用 Docker DNS resolver 延迟解析

### ChatBridge WebSocket 连通
- **根因**: `index.html` 硬编码 `ws://172.18.0.1:8850`，WSL2 下 Windows 浏览器不可达
- **修复**: 改为 `ws://localhost:8850`，利用 WSL2 端口转发
